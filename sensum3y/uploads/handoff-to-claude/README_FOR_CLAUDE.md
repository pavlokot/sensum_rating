# Sensum Landing Handoff

Це поточна версія статичного landing / impact report для БФ «Сенсум» за 2023-2026.

## Файли

- `index.html` — структура сторінки.
- `styles.css` — поточна візуальна система.
- `script.js` — база подій, хронологія, великі блоки, галереї.
- `assets/photos/` — усі актуальні фото, які вже прив'язані до подій.

## Завдання для фінальної верстки

Зробити сучасний, зрозумілий і стильний landing / impact report, використовуючи шрифти й стилі, які вже підготовлені в Claude Design.

## Жорсткі правила

- Не вигадувати нових фактів, дат, сум, підрозділів, передач чи партнерів.
- Не змінювати зміст подій по суті.
- Усі наявні події зберегти.
- Усі ключові акценти зберегти: `53 ОМБр`, `45 ОАБр`, `БпАК «Мара»`, `Птахи Мадяра`.
- Фото залишити чорно-білими.
- Соціально-спортивні блоки залишити з жовтим акцентом.
- Верхнє меню не потрібне.
- Розділ `Засновники фонду` потрібен, але `9 засновників` не виносити в блок здобутків / метрик.
- В кінці має бути місце для Facebook, Instagram і QR-коду поточного збору.

## Візуальні побажання

- Не вивалювати всі ключові великі блоки поспіль на початку: розподілити великі акценти по всьому сайту.
- Балансувати великі story-блоки з компактними картками хронології.
- Фото мають бути акуратно кадровані під різні формати: горизонтальні, вертикальні, квадратні.
- Важливе фото `Птахи Мадяра` має лишатися більшим за звичайні фото в цьому блоці.
- Не допускати накладання текстів, обрізаних заголовків і горизонтального overflow.
- Довгі українські слова у заголовках мають переноситися всередині карток.

## Перевірка перед фіналом

Обов'язково перевірити:

- Desktop ширина приблизно `1280-1440px`.
- Mobile ширина приблизно `390px`.
- Немає горизонтального скролу.
- Усі фото завантажуються.
- Усі фото чорно-білі.
- Ключові події та дати залишились на місці.
- Соціальні / спортивні картки виділені жовтим.
- Немає текстів-заглушок типу `[уточнити]`.

## Локальний перегляд

Якщо відкривати як статичний сайт:

```bash
python3 -m http.server 4178 --bind 127.0.0.1
```

Потім відкрити:

```text
http://127.0.0.1:4178/index.html
```
